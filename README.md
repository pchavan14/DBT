DBT project to Analyze airbnb's listings in Berlin region

### Using the starter project

Try running the following commands:
- dbt run
- dbt test

Re-run below commands when you restart terminal
virtualenv venv
. venv/bin/activate
rehash
pip install dbt-snowflake
which dbt
